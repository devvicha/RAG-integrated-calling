# Savings Account Calculator

## Overview
Calculate the growth of your savings account based on:
- Initial deposit amount
- Monthly contribution
- Annual interest rate
- Time period (in months)

## Formula
The calculator uses compound interest formula:
- Future Value = Initial Deposit × (1 + r)^n + Monthly Deposit × [((1 + r)^n - 1) / r]
- Where r = monthly interest rate, n = number of months

## Minimum Requirements
- Minimum initial deposit: LKR 1,000
- No minimum for monthly deposits

## Features
- Calculates total savings value
- Shows total interest earned
- Provides results in natural Sinhala pronunciation

## Usage
Ask: "මට ඉතිරිකිරීම් ගිණුමක් වර්ධනය ගණනය කරන්න පුළුවන්ද?"
